
### ✅ **Bases de datos y almacenamiento**
- **SQL Server**: Motor relacional principal para datos estructurados.
- **Azure SQL Database**: Versión en la nube de SQL Server.
- **Azure Data Lake Storage**: Almacenamiento masivo para datos no estructurados.

---

### ✅ **ETL y procesamiento**
- **SSIS (SQL Server Integration Services)**: Para extracción, transformación y carga (ETL) en entornos on-premises.
- **Azure Data Factory**: ETL en la nube, orquestación de pipelines y conexión con múltiples fuentes.
- **Azure Synapse Analytics**: Procesamiento masivo y análisis de datos (Big Data + Data Warehouse).

---

### ✅ **Modelado y análisis**
- **SSAS (SQL Server Analysis Services)**: Modelos OLAP y tabulares para análisis multidimensional.
- **Power BI**: Visualización, dashboards y análisis interactivo.
- **Azure Analysis Services**: Versión cloud de SSAS.

---

### ✅ **Reporting**
- **SSRS (SQL Server Reporting Services)**: Informes paginados tradicionales.
- **Power BI Report Server**: Informes paginados + dashboards modernos en entornos locales.

---

### ✅ **Big Data y Machine Learning**
- **Azure Databricks**: Procesamiento distribuido y análisis avanzado.
- **Azure Machine Learning**: Entrenamiento y despliegue de modelos ML.

---

### ✅ **Integración y gobierno**
- **Azure Purview**: Catalogación y gobierno de datos.
- **Azure Event Hub / Azure Stream Analytics**: Procesamiento de datos en tiempo real.

---
